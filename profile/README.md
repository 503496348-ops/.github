# 503496348-ops · AtomCollide

AI 产品、自动化工具与 Agent Skill 项目的公共矩阵。

## 我们在做什么

- 把可复用的 AI 工作流打磨成可运行、可验证、可维护的公开项目。
- 为每个项目建立统一的安全、贡献、Issue/PR 和质量门禁。
- 将内部 Skill 中适合公开的部分升级为可赞助 Project；不适合公开的能力保留在内部。

## 项目治理

本账号仓库采用统一公共治理规范：Issue/PR 模板、安全报告、贡献流程和支持说明由 `.github` 默认治理仓提供。每个产品仓可按业务需要覆盖默认文件。
